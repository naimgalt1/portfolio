# Naim Galet Naieb — Portfolio

Portfolio personal desarrollado con React, Tailwind CSS, Framer Motion y Three.js.

## Stack
- React 19
- Tailwind CSS
- Framer Motion
- @react-three/fiber
- FastAPI + MongoDB (backend del formulario de contacto)

## Instalación
```bash
yarn install
yarn start
```
